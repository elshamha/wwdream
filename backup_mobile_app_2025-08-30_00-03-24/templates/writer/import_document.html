Supported Formats

  • PDF Documents (.pdf)
  • Microsoft Word (.docx, .doc)
  • Plain Text (.txt)
  • HTML Files (.html, .htm)
  • Rich Text (.rtf)
  • OpenDocument (.odt)
  • Google Docs (export as .docx first)
Google Docs Instructions
  1. Open your Google Doc
  2. Go to File → Download → Microsoft Word (.docx)
  3. Upload the downloaded .docx file to the import page
  4. All formatting, images, and structure will be preserved!
Import Tips & Guidelines
Best Practices
  • Use descriptive titles for easy identification
  • Assign to a project for better organization
  • Check "Preserve Formatting" for styled documents
  • Enable metadata extraction for detailed information
  • Test with smaller documents first to understand the conversion process
  • Save your original files as backups before importing
Limitations
  • Maximum file size: 50 MB
  • Complex layouts may not convert perfectly
  • Images and tables are converted to text descriptions
  • Some formatting may be lost during conversion
  • Password-protected files cannot be imported
  • Embedded objects may not be preserved
Pro Tips
For Better Results
  • Clean up your document formatting before export
  • Use consistent heading styles (H1, H2, H3)
  • Remove unnecessary page breaks and spacing
  • Use standard fonts when possible
Troubleshooting
  • If import fails, try saving as a different format
  • For large PDFs, consider splitting into chapters
  • Check file permissions if upload is blocked
  • Contact support for files over 50MB
